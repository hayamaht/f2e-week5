1：廣告版位：請設計出不會讓使用者感到閱讀障礙的廣告瀏覽體驗。
2：圖片載入：該如何切換到下一頁，儘量減少使用者等待時間。卻又不是全部頁面直接載入造成伺服器 Loading，端看於各位開發者考量設計出滿足兩者需求的前端介面。
